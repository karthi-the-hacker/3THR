# 3THR
Find and Filter all open ports 

3THR tool installation steps:

1. cd ~
2. cd tools
3. git clone https://github.com/karthi-the-hacker/3THR.git
4. cd 3THR
5. chmod +x 3THR.sh
6. chmod +x .banner
7. ./3THR.sh
